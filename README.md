# 大山 CRM v3.0

企业级多团队协作与营销管理系统，基于现代化全栈架构复刻与重构。

## 🚀 核心特性

- **Salesforce 风格 UI**: 采用企业级 Lightning 设计语言，极致的视觉交互体验。
- **全流程业务闭环**: 涵盖订单管理、产品录入、财务对账、员工管理等核心模块。
- **批量处理矩阵**: 支持订单导出 CSV、运单号批量导入、物流自动同步及财务数据对账。
- **高性能架构**: 后端基于 Node.js (Express)，前端采用 React 18 + Tailwind CSS。
- **预置演示数据**: 内置 10+ 条涵盖华为、腾讯等企业的模拟业务订单及财务流水。

## 🛠️ 技术栈

- **前端**: React 18, Tailwind CSS, Lucide Icons
- **后端**: Node.js, Express, JWT (Authentication), Bcrypt (Security)
- **数据库**: 内存数据库 (带 Seeder 自动初始化)
